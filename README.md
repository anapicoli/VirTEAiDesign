# VirTEAi Design

Design, prototipação e experiência do usuário do projeto **VirTEAi**.

## Sobre o Projeto

O **VirTEAi** é uma plataforma digital desenvolvida para apoiar pessoas com **Transtorno do Espectro Autista (TEA)** por meio de experiências interativas, acessíveis e centradas no usuário.

Este repositório concentra todos os artefatos de **UX/UI Design**, incluindo pesquisas, fluxos de navegação, wireframes, protótipos de alta fidelidade, sistema de design e componentes visuais utilizados no desenvolvimento da solução.

O objetivo é criar uma experiência intuitiva, acolhedora e adaptada às necessidades sensoriais e cognitivas dos usuários, promovendo autonomia, aprendizado e desenvolvimento de habilidades sociais.

---

## Objetivos do Design

* Garantir acessibilidade e inclusão digital;
* Reduzir sobrecarga visual e cognitiva;
* Facilitar a navegação por meio de interfaces intuitivas;
* Promover uma experiência amigável para usuários com TEA;
* Manter consistência visual em todas as plataformas;
* Apoiar a evolução contínua do produto através de testes e validações.

## Princípios de Design

### Acessibilidade em Primeiro Lugar

Interfaces projetadas considerando diferentes perfis de usuários, priorizando clareza, legibilidade e previsibilidade.

### Simplicidade Cognitiva

Redução de distrações visuais, organização hierárquica das informações e fluxos simplificados.

### Consistência

Utilização de componentes reutilizáveis e padrões visuais padronizados para melhorar a experiência de uso.

### Design Centrado no Usuário

Decisões baseadas nas necessidades reais dos usuários e especialistas envolvidos no contexto do TEA.

---

## Design System

O Design System do VirTEAi define:

* Paleta de cores;
* Escala tipográfica;
* Espaçamentos;
* Componentes reutilizáveis;
* Padrões de interação;
* Diretrizes de acessibilidade.

Seu objetivo é garantir escalabilidade, consistência e manutenção eficiente da interface.
